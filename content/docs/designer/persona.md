# ペルソナ

Description

ターゲットユーザーをひとりの人間にみたて、チームで認識を合わせる。

- 以下を記入
    - FACT
    - BEHAVIOR
    - NEEDS & GOAL
    - 名前と簡単なイラスト
- 現時点での理解（アサンプション）を可視化する
- お互い質問して、理解を深める
- ペルソナの名前は以降で常に使う

Tips

- デザイナーだけで作業せず、PMsやユーザーをよく知る人と一緒にワークする
- 1時間程度で仕上げる。
- 正しいか正しくないかよりも、全員の認識を可視化することが大事。
- 「今わからないこと」は貴重な発見
- プロダクトに関係のないことは書かない
    - 例えばBtoBシステムの場合、趣味や家族構成はあまり関係ないことが多い。
- あらたな事実や誤りに気づいたら随時修正していく。
