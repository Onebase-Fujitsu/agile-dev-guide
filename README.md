## How to develop

We are currently not accepting contributions from outside Fujitsu.

1. install Go  
   download [go sdk](https://golang.org/dl/) and install.
2. install Hugo
    ```shell
    brew install hugo
    ```
3. launch development server
    ```shell
    hugo server -D
    ```
